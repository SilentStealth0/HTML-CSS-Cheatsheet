About Tables

Name Syntax Purpose
Table <table>, </table> Creates the table that will contain the data
Table Row <tr>, </tr> To enter data, first step is to make a table row
Table Data <td>, </td> Each cell must also be defined, td allows you to add data
Spanning Columns colspan When data spans more than 1 column, use this with an integer (greater than 1) to denote the number of columns it spans
Spanning Rows rowspan Used for data that spans multiple rows; accepts an integer to denote the number of rows it spans across.
