About Tables

Name Syntax Purpose
Table <table>, </table> Creates the table that will contain the data
Table Row <tr>, </tr> To enter data, first step is to make a table row
Table Data <td>, </td> Each cell must also be defined, td allows you to add data
Spanning Columns colspan When data spans more than 1 column, use this with an integer (greater than 1) to denote the number of columns it spans
Spanning Rows rowspan Used for data that spans multiple rows; accepts an integer to denote the number of rows it spans across.

Semantic HTML

Name Syntax Purpose
Header <header>, </header> Container usually for either navigational links or introductory content
Nav <nav>, </nav> Used to define a block of navigation links
Main <main>, </main> encapsulate the dominant content within a webpage.
Footer <footer>, </footer> Content at the bottom of the subject information
Section <section>, </section> Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
Article <article>, </article> Holds content such as articles, blogs, comments, magazines, etc.
Aside <aside>, </aside> Used to mark additional information that can enhance another element but isnâ€™t required in order to understand the main content.
Figure <figure>, </figure> Element used to encapsulate media such as an image, illustration, diagram, code snippet, etc. It will encapsulate the img tag
Figcaption <figcaption>, </figcaption> Element used to describe the media in the figure tag.
Audio <audio>,</audio> Used to embed audio content into a document. Use src to link source.
Video <video>, </video> Element makes it clear that a developer is attempting to display a video to the user.

Selectors & Specificity

Name Syntax Purpose
Type p or h1 or main or td The type selector matches the type of the element in the HTML document.
Universal * Universal selector selects all elements of any type
Class .name or .container Class attribute is one of the most common ways to select an element. Can select multiple classes.
ID #large-title or #img-background used to select a single element since ID's can only be used once per page
Attribute [href] or [src] Used to select HTML elements that already contain attributes
Pseudo-class p:hover / :focus,:visited,:disabled, and :active Helps give a different state to help with certain user interaction and site navigation
Specificity ID > Class > Type The order by which the browser decides which CSS styles will be displayed
Chaining h1.special or .main p Be able to chain multiple selectors to select specific elements such as only selecting h1 elements with a class of special
Descendant Combinator .main-list li Select elements that are nested withing other HTML elements like selecting all the li elements descendants from an ul class 'main-list'
Multiple Selectors h1, .menu Possible to add CSS styles to multiple CSS selectors all al once

Colors

Name Syntax Purpose
Hexadecimal #FFF, #A52A2A, #00FFF Hex color begins with a hash character (#) then followed by 3 or 6 characters.
RGB Colors rgb(23, 45, 23) Each value represents a color component, the first represents the amount of red, the second represents amount of green, and last represents blue.
Close to how computers represent colors.
Hue, Saturation, and Lightness hsl(120, 60%, 70%) Hue which refers to an angle on the color Wheel.
Saturation is the intensity or purity of the color where 100% makes the color richer and 0% makes the color grayer.
Lightness refers to how light or dark the color is. Halfway (50%) is normal lightness. Towards 100% makes the color lighter or closer to white. Towards 0% makes the color darker, closer to black.
Opacity and Alpha hsla(34, 100%, 50%, 0.1) or rgba(234, 45, 98, 0.33) The Alpha value controls the amount of the background to mix with the foreground or the opacity of the element.
When below 1, any color behind will be blended in.
For hsla, the first 3 values vales are the same as Hue, Saturation, and Lightness but the fourth value is alpha.
For rgba, the first 3 values work the same as RGB colors while the last value is the alpha.
